# Repo Tool-Use Router

## Purpose
This page is a historical scan router from the repo's growing tool/probe estate into the wiki harness. Current count-bearing status belongs in the spec mirrors linked below.

It answers a narrower question than [[tooling-status]]:

- not merely which tools are installed;
- not merely which tools appear in files;
- but what current repo receipts allow each tool to falsify, witness, route, or support in the current constraint-admissibility program.

## Evidence boundary
The inventory behind this page scanned repo text/code and wiki pages for tool mentions. Counts are route-finding signals, not proof of load-bearing use.

- mention count -> installed;

- tool success -> canonical by process;
- provider agreement -> sim evidence.
