# Mimetic Meme Manifold Harness

## Overview
This page describes the harness-facing role of the mimetic meme manifold: a salience-shaping language layer used to load a model into the right conceptual space.

## Main points
- The harness uses a project-specific dictionary to shape salience.
- Language control is operational, not decorative.
- The macro layer stays separate from lower-layer admissibility tests.
- The harness should support booting and group-level coupling without replacing the base math.
