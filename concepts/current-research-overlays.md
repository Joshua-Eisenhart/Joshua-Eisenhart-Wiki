## How Sources Feed the Stack
Sources feed into the research through a layered ingestion process:
1. Raw source artifacts land in `raw/articles/new-docs/` for imported docs and `raw/papers/open_access/` for paper-intake PDFs, extracted text, and manifests.
2. Preprocessed digests extract key structures and claims
3. Sim code tests specific claims against the [[constraint-on-distinguishability-full-math|constraint surface]]
4. Results are recorded in JSON artifacts
5. The [[current-preaxis-status-and-ordering-note|status ledger]] tracks what has been recorded, rerun, or kept open at the current evidence level

6. The overlay layer attaches the relevant external reference pages so future research does not have to rediscover the support structure from scratch.

Overlay fence:
- this routing layer should not silently upgrade packet/result existence into stronger admission
- use it to find the right narrower page, not to bypass the status/provenance controls named in the harness/controller docs
