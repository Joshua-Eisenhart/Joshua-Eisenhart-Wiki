# Legacy Speculative Theory

## Overview
Older synthetic theory notes that mix cosmology, governance, and personality models. Read them as historical speculation, not current authority.

This page is most useful as a warning label: these sources often fuse domains that the live wiki now tries to keep distinct.

## Main fusion pattern
The recurring legacy blend is:
- cosmology / foundational metaphysics
- governance / social order
- personality / emotional grammar
- world-model / memory ideas
- civilization-scale ambition

That is why the cluster can be rich but drift-prone. The current wiki should preserve the fusion historically while routing readers outward into more specific pages.

## Role in the live wiki cluster
- **Strongest use:** caution page for fusion-heavy legacy synthesis when cosmology, governance, personality, and world-model language start collapsing into one blob.
- **Weak use:** not the right first-stop page for provenance, not the branch-separation page, and not a dev-facing translation surface.
- **Authority boundary:** warning-label summary only. It names the drift pattern and routes readers outward; it does not resolve the fused claims into current doctrine on its own.

1. Read [[current-docs-vs-legacy-framing]] first for the current-versus-legacy boundary.
