# Aligned Sim Backlog Source Digest

## Overview
Source digest for the aligned sim backlog and build order. Covers collapse classes, dependency order, and next aligned sims.

## Key Points
- Build small sims as legos
- Start from root constraints
- Compose upward from admitted or fenced pieces
- Include negatives in every sim
- Emit witnesses/artifacts
- Finish pre-Axis tiers before Axis-entry work

## Dependency Order
Constraints -> carrier -> geometry -> transport -> chirality -> negatives -> bridge -> cut -> entropy -> kernel -> graph writeback.
