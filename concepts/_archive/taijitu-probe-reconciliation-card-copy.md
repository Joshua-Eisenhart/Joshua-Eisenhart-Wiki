# Taijitu Probe Reconciliation Card

- **Status:** Working support surface. Controller-grade reconciliation card.
- **Scope:** Reconcile the explicit taijitu symbolic packet with the current lower-axis atlas, the Jung/IGT packet, the older yin-yang probe, and the fenced 64-slot scaffold surfaces.

## 1. Strong Alignments

- `Axis 0` strongest shared read: black-versus-white symbolic split is real, but the lower-stack math anchor is still `b_0 = sign( cos(2 eta) ) = sign( r_z )`, with the final `Phi_0(rho_AB)` bridge and cut still open.
- `Axis 1` strongest shared read: the current lower-stack branch split is `{ Se, Ni }` versus `{ Ne, Si }`. The dot-and-teardrop layer is a usable symbolic witness for that split.
- `Axis 2` strongest shared read: the lower-stack anchor is direct frame `{ Se, Ne }` versus conjugated frame `{ Si, Ni }`. Dots-versus-teardrops remains a symbolic witness, not the primary frame law.

- `Axis 6` strongest shared read: judging-first versus perceiving-first order is directly readable from the ordered tokens and is stronger than older boundary-side language.
- The 64-slot scaffold remains usable as addressing: `8 terrains x 8 signed operators = 64` is still a valid structural scaffold if it stays fenced as representation.

- `Axis 0` mismatch: the taijitu packet can safely say black versus white and enclosing-circle drive, but that language must not replace the lower-stack `Axis 0` bridge/cut family.
- `Axis 1` mismatch: the older yin-yang probe frames `Axis 1` as interpenetration via off-diagonal coherence, while the current lower stack uses the derived branch split `{ Se, Ni }` versus `{ Ne, Si }`.
