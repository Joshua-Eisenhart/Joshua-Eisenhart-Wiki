# Mimetic Meme Manifold Boot Dictionary Digest

## Overview
Source digest for the boot dictionary aspect of the mimetic meme manifold. Covers the specific vocabulary that loads an LLM into the right thinking space.

## Key Points
- A harness is partly a language-loading problem
- Each fresh LLM instance needs a dictionary of aligned language
- The dictionary makes the right things salient
- Constraint engineering needs its own dictionary
- Each project builds a custom dictionary on top to shape saliency
- Control over vocabulary is operational power, not decoration

## Boot Dictionary Elements
The boot prompts in [[boot-prompt-templates]] implement the dictionary: "constraint on distinguishability" = primitive; a=a iff a~b = identity requires contrast; M(C) = constraint surface; no primitive causality; no "X IS Y" -- use structural correspondence.
