# Skill Source Corpus

## Overview
An umbrella corpus of docs, repositories, method notes, and runtime substrate material intended to feed future skill and operator building.

## Main points
- Treats the corpus as a growing collection rather than one canonical source.
- Distinguishes raw source materials from derived skill outputs.
- Emphasizes bounded selection, integration tracking, and corpus-driven build planning.
- Serves as a front door for skill clusters and runtime adapters.
