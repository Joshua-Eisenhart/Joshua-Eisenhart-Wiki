# Lean 4 Proof Assistant Reference

## Overview
Lean 4 is a planned proof tool in the stack rather than a currently installed live layer. It matters because it represents the strongest machine-checked proof direction currently named in the tooling surfaces.

## Best-fit jobs in this stack
- formal theorem/proof objects
- machine-checked proof scripts
- stronger long-horizon proof surfaces than SAT/SMT alone

## Why it matters here
The project already uses bounded proof pressure from SMT and symbolic tools. Lean 4 would extend that into explicit theorem-proof objects when or if it becomes a live integrated layer.

## Honesty fence
This is a planned-tool page. It does not imply Lean 4 is currently installed or compiled into the live pipeline.
