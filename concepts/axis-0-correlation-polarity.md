# Axis 0 Correlation Polarity

## Overview
Axis 0 is defined as correlation persistence polarity under perturbation: whether correlation diversity spreads or gets damped when a system is disturbed.

## Main points
- Allostatic: correlation diversity increases under perturbation.
- Homeostatic: correlation deviation is suppressed under perturbation.
- The document treats its formulas as testable options rather than canon.
