## The core claim

Perception is not object recognition. Objects are not primitive. They are functional, mimetic story-units — boundaries drawn by use, attention, and desire, and they hold only while independent perspectives keep converging on them. This layer sits UNDER admission gates: gates decide what evidence is admitted; object formation decides what things exist for evidence to be about.

Three presumptions this removes (the ones nearly all software, including gate architectures, smuggle in): pre-existing objects, universals, and equality signs. In CR terms these are exactly F01/N01/`a=a iff a~b` applied to perception.

## How objects get created (human, then machine)

Humans: **handling** (grasp, rotate, pound, push — a candidate that gets cheaper to predict as it is worked starts behaving like a thing) + **mimetic convergence** (we only see what others see; a private percept is a hallucination candidate; objects become real through independent eyes and hands converging, then get stored in a shared vocabulary). One perceiver cannot create an object even in principle — objecthood IS convergence across independent perspectives. That is why perception is a mesh function, not a node function.

Machine loop (prediction-first, per the holodeck model): project a candidate story-unit → handle it → check whether independent probe paths converge → earn (confirmation hash into the shared vocabulary) or kill (anti-hash into the graveyard) → desire/attention re-weights what gets projected next. Desire is the gradient — Axis 0 at the cognitive scale; it selects what gets projected and probed, never what survives.

## The formal replacement for the equality sign

Every "=" decomposes to "~ under a named probe family M". An earned object is a probe-quotient cell that passes three tests it didn't need to pass to be a mere cell:

1. Handling — prediction error falls as it is worked; noise gets more expensive.
2. Convergence — the cell survives when another node's probe family is joined (collective indistinguishability).
3. Persistence + cheap recall — it holds across time and re-confirms cheaply from context.

Content hashes are storage identity, not world identity — one probe among many, never identity itself.

## Memory (not flattened — codex1-cited from the holodeck source)

Memory is `generate -> compare -> confirm -> refine`, not `archive -> fetch -> replay`. Recall is a walk through a cue field, not a single lookup: confirmation keys stack and chain, so local cues can walk the system toward remote structure. Failed candidates persist as anti-hashes so the same hallucination can't re-enter by another route. Identity is world-coupled: an object stays the-same-thing because it is embedded in a live correction loop, not because a profile is stored — a candidate formed only against replayed fixtures is `static_only` and cannot reach full objecthood.

## Why the mesh (v3.2's two constraints)

The mesh only works as an object-factory under two load-bearing constraints from Leviathan v3.2:

1. Witnessed self-accounting as the membership act (the Ugly Mirror): a node must face its own accounting, witnessed by peers, to participate at all. Without honest nodes, convergence is poisoned.
2. Dynamic delegated hierarchy, power down / dictionary held: all cooperation through revocable, layered, veto-retaining delegation; the apex holds only the shared vocabulary (the MMM). Since the dictionary is the object store, ontology control IS object control. Trust is delegated and revocable on empirical outcome, with base veto and single-point accountability — which is what keeps an attention/desire layer from becoming hype-driven.

## Loss-function connection

A loss function is a formalized desire; a task harness is a directed gaze; unknown-unknowns are the gaze gap (invisible to a single harness-designer by construction). The mesh generates the next loss function — cross-node attention convergence surfaces which harnesses should exist and retires ones no gaze converges on. Objects are congealed loss functions: story-units many harnesses kept finding cheap to predict and worth attending to.
