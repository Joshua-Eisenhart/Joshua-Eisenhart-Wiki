# Ladders, Fences, and Admission: Reference

## Overview
Extracted from v5 docs. Covers the 19-layer pre-entropy ladder, clean build chain, mass table of required sims, and pre-Axis admission pipeline. Despite "ladder" language, the layers are resolution levels, not ontological floors.

## 19-Layer Pre-Entropy Ladder
Entropy is layer 17. Layers 1-16 are prerequisites. Key layers:
- Layer 1-2: Root constraints, probe-relative identity
- Layer 3: Admissible domain M(C)
- Layer 4: Finite carrier H=C^2
- Layer 5: Hopf carrier geometry S^3 -> S^2 with nested tori
- Layer 6-7: Nested torus stratification, Weyl extraction
- Layer 8-10: Chiral density, loop laws, engine family split (Type 1 vs Type 2)
- Layer 11-13: Placement law, negatives on geometry, entangling operators
- Layer 14-16: Joint cut state rho_AB, bridge family, shell/history support
- Layer 17: Signed/unsigned entropy (S, S(A|B), I, I_c)
- Layer 18: Axis 0 kernel
- Layer 19: Dynamics on entropy

## Clean Build Chain
constraints -> M(C) -> finite carrier -> S^3/Hopf -> nested tori -> (psi_L, psi_R) -> loop laws -> engine family -> rho_AB -> Xi_family -> S, I, S(A|B), I_c

## Mass Table: What Must Be Simmed Before Entropy Is Honest
Each domain needs simulation before entropy can be admitted:
- constraints: falsifications from altered root assumptions
- carrier: finite carrier alternatives
- geometry: Hopf, nested torus, transport, holonomy
- chirality: L/R Weyl extraction and failure modes
- loop law: fiber/base distinction, swap negatives
- engine family: Type 1/2 split, dominance allocation
- entanglement: joint state formation, preservation, degradation
- bridge: point/shell/history comparisons
- negatives: scrambled torus, no chirality, swapped loop

## Pre-Axis Admission Pipeline
Six phases: A (define) -> B (ground) -> C (refine) -> D (classify) -> E (embargo) -> F (elevate). Each phase has formalized actions, sim requirements, and attack surfaces.

## Admission Checks and Classical Leakage
Six admission checks: Constraint alignment (no ontology smuggling), QIT alignment (no classical leakage), Same-carrier honesty (no fake necessity), Negative support (no arbitrary promotion), Placement clarity (no diagnostic-as-law), Axis embargo (no unstable substrate). Six classical leakage types: Continuum (smooth flow without finite-carrier), Metric (geometry before QIT grounding), Entropy (primitive entropy), Dynamics (classical trajectory), Flux (change renamed as flux), Closure (correlation as necessity). (from LADDERS_FENCES_ADMISSION_REFERENCE.md)

## Admissibility Fences
BC04-BC12 bans: identity, equality, global order, closure by default, probe-relative ID, probability primitives, metric/coordinates, optimization, smuggling. T1-T8 topology fences: scoped compatibility, non-metric adjacency, local only, path non-collapse, anti-identity, anti-scalarization, anti-geometry import, anti-smoothness. These fences enforce [[constraint-on-distinguishability|F01 and N01]] at every layer. (from LADDERS_FENCES_ADMISSION_REFERENCE.md)
