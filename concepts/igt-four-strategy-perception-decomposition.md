# IGT Four-Strategy Perception Decomposition

**IGT = Irrational Game Theory** (source: `apple axes terrain operator math.md`; `igt-to-qit-engine-genealogy`). "Irrational" = Jung's *perceiving* functions (Se/Ne/Ni/Si), the perception/strategy layer **before** rational judgment. The four perceiving functions are proposed as four **perception policies** â€” four ways an organism projects viability structure onto the world before rational judgment arrives â€” not personality labels at the base.

The chain this page decomposes:

```text
evolutionary selection patterns
  -> embodied strategies / emotional action-readiness
  -> projected perception
  -> Jung irrational/perceiving functions (Se/Ne/Ni/Si)
  -> IGT win/lose terrains
  -> QIT engine terrains (flow/topology classes)
```

This page separates what is **LOCKED** (source + math grounded), what is **INTERPRETIVE** (owner-source candidate, per-row falsifier), and what is **FRAME** (philosophical synthesis, not proved). Claim ceiling: candidate cross-domain grammar; NOT admitted neuroscience, NOT a solved theory of consciousness. The ceiling is **NOT** "personality types map to physics" â€” it is "evolutionary viability patterns -> embodied emotional strategies -> perception policies -> Jung irrational functions -> IGT terrains -> QIT formalization."

## Core principle: REGIME, not LABEL

**Each quadrant is a perception/policy REGIME. Emotion, Jung function, 4F, evolutionary strategy, hormone, and QIT terrain are different READOUTS of that one regime â€” not the regime itself.** This is the "different faces of the same thing" frame (`a=a iff a~b`): the regime is the object; the columns are probes of it. Do NOT collapse into a brittle dictionary ("Se = this emotion, Ne = this emotion") as if the labels were primitive. `Se/Ne/Ni/Si` are the **legacy naming surface** for four perception-policy classes, not personality labels at the base. The reason this matters: if the columns are readouts of one regime, a wrong column-value is a bad readout (fixable); if the columns are the definition, a wrong value breaks the claim. Hold them as readouts.

Hoffman resolution (the move this page makes): Hoffman dethrones spacetime/objects but then posits "conscious agents" as a new fundamental substrate (a Platonic move). This decomposition needs no new substrate: the **same constraint-admissibility pattern at every scale** (physics / evolution / perception); consciousness = the pattern perceiving itself through evolved strategy-projection. "Evolution selects for its own pattern in the root of consciousness." The self-undermining worry (if perception is not true, how do we know anything?) dissolves: knowing was never truth-copying â€” it is constraint-survival (the ratchet = wide explore + graveyard + survivors = Hume's induction-by-custom).

## 4. Falsifiers / negative controls (per row + global)

**Global falsifier:** if the four strategy groups do NOT uniquely / stably map onto distinct perception/emotion/action-readiness regimes, the 4-fold is a good symbolic overlay but NOT a load-bearing decomposition.

The four groups should remain separable on these observable dimensions (if they collapse, the decomposition weakens):

1. attentional target: threat / opportunity / stability / contact
2. action tendency: approach / withdraw / freeze / bond / explore / exploit
3. time horizon: immediate / future / past-stability / deep trajectory
4. body state: SNS / PSNS / mixed / regulation mode
5. social strategy: competition / cooperation / sacrifice / preservation
6. FEP role: epistemic exploration / pragmatic preservation / risk minimization / policy switching

**Per-row falsifier (the test):** shuffle the assignment for one row (e.g. give `Ne` the `Si` body-state/selection/emotion) â€” if no observable dimension distinguishes the shuffled from the canonical assignment, that row's binding is not load-bearing.
