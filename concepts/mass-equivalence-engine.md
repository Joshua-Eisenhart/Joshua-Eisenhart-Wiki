# Mass Equivalence Engine

## Core claim
Many domains can be represented by the same operational engine:
- state carrier
- probe family
- operator/channel dynamics
- admissible geometry
- information functional
- constraint manifold

The point is not that the formulas are identical. The point is that they can often be cast into a common representation.

## Structural classes
- State-carrier class: density matrices, rays, coherence vectors
- Geometry class: trace distance, fidelity, Fubini-Study, QFI, Berry structure
- Correlation class: mutual information, conditional entropy, coherent information, entanglement measures
- Operator/dynamical class: channels, Kraus maps, Lindbladians, commutators
- Compression/factorization class: PCA, SVD, Schmidt decomposition, low-rank truncation

## Use in the system
This doc is the bridge between the lower-level constraint surface and the cross-domain mathematics that can be admitted later.
