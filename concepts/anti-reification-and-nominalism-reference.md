# Anti-Reification and Nominalism Reference

Nominalism denies that universals exist independently of particular instances. Anti-reification denies that abstract categories are real objects. Together: labels are tools for grouping, not discoveries of deep structure.

Support fence: this page is an operational language and commitment discipline for the wiki/harness. It should not be read as a standalone metaphysical proof, and examples below need the named probe, constraint set, and receipt before they become formal claims.

## Core Positions

### Classical Nominalism (Ockham, Berkeley, Hume)

**Ockham's Razor:** Do not multiply entities beyond necessity. If "entanglement" can be explained as "states that survive PPT tests," do not posit "entanglement" as a separate entity.

**Berkeley:** To be is to be perceived (esse est percipi). Applied operationally: a property is admitted for the current formal use only insofar as it is detected by a probe. "The tree in the forest" problem is used here as probe discipline, not as imported idealist doctrine.

**Hume:** Causation is constant conjunction, not necessary connection. Applied: "L4 kills absolute measures" means "absolute measures are absent after L4," not "L4 causes their destruction."

### Modern Nominalism (Quine, Goodman)

**Quine:** Existence is what the existential quantifier ranges over. Applied: "there exist 28 irreducible families" means "∃x (x ∈ surviving_set ∧ ...)" — a formal claim about a set, not an ontological claim about reality.

**Goodman (New Riddle of Induction):** "Grue" (green before t, blue after) is as legitimate as "green." Applied: the system's families are "grue-like" — they are defined by which constraints are active, not by intrinsic properties. The same state can be "admissible" under one constraint set and "excluded" under another.

### Structuralism Without Universals (Hellman, Shapiro)

**Hellman's modal structuralism:** Mathematical objects are positions in structures, not independent entities. Applied: density matrices are positions in the CPTP-constrained structure, not "things" with intrinsic quantum properties.

## Anti-Reification Traps

### Trap 1: Category as Object

**Reified:** "Entanglement is a resource that can be distilled."
**Anti-reified:** "States classified as entangled by concurrence/negativity can be transformed into states with higher concurrence/negativity under LOCC."

The difference: "entanglement" is not an object being moved around. It is a classification label applied to states under specific operations.

### Trap 2: Aggregate as Entity

**Reified:** "The constraint manifold M(C) has curvature."
**Anti-reified:** "States surviving constraint set C form a set on which the Bures metric has nonzero Riemann tensor components."

M(C) is not an entity with properties. It is a shorthand for "the set of x such that x survives C."

### Trap 3: Trend as Force

**Reified:** "The cascade drives toward irreducibility."
**Anti-reified:** "When constraints L0 through L7 are all active, the surviving set has 28 members. Under weaker constraint sets, the surviving set is larger."

There is no "driving." There is only: which constraints are active, and which structures survive.

## Connection to This System

### Identity as Distinguishability Class

The system's core claim: identity is not intrinsic. Two states are "the same" iff they are indistinguishable under the active probes and constraints. Change the probe -> identity changes. Change the constraints -> identity changes.

This is nominalism applied as an operational physics discipline: for a given formal use, the probes and constraints serve as local identity conditions. Broader metaphysical denial is not doing the proof work; the active probe/constraint receipt is.

### Constraint-Admissibility as Anti-Reification

For formal claims, the system should avoid bare "X exists" language and prefer "X is admissible under constraint set C." This is anti-reification built into the formalism: admissibility is a relation between an object and a constraint set, not a property of the object.

### The 28 Families Are Not Universals

A receipt-bound "28 irreducible families" claim would not be a Platonic form claim. It would be a claim about equivalence classes under the active constraints, with the exact result receipt and constraint set named. Under different constraints, different equivalence classes could form.

## Key Results

1. **Goodman's grue problem has no solution in universals**: any classification is arbitrary without specifying the projectible predicates. For this formal use, the system's constraint set supplies the projectibility specification.
2. **Quine's ontological relativity**: reference is indeterminate without a background theory. For this formal use, the system's constraint set supplies the background theory.
3. **Structuralism without objects**: mathematical structures are defined by their positions and relations, not by the objects that occupy them. The system's computational graph is relational, not object-based.
