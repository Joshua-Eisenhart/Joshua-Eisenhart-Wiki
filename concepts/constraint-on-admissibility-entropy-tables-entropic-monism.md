# Constraint on Admissibility, Entropy Tables, Entropic Monism

## Overview
This compound reference note combines the admissibility constraint doctrine, entropy table formatting guidance, and entropic monism clarification. It is a legacy provenance document; use the cleaner canonical pages for current work.

## Entropic Monism: Core Claim
The primitive substance is **constraint on distinguishability**, not generic entropy. The downstream identifications are:
- matter = constraint
- energy = constraint flow
- geometry = constraint compatibility
- dynamics = constraint-respecting transformation
- information = constraint bookkeeping

### What Entropic Monism Actually Means
- "Entropic monism" is the **doctrine name**
- "Constraint on distinguishability" is the **actual primitive claim**
- "Entropy" is a **later admissible measure** of distinguishability under constraint
- The sloppy claim that "everything is entropy" is rejected; the deeper claim is that one primitive substance exists: constraint on distinguishability

## Entropy Tables: Formatting Notes
The content of the entropy tables is verified correct, but formatting issues existed:
- Multiline display equations inside narrow cells
- Repeated long equations that should be factored into named rows
- Inconsistent use of inline vs display math
- Some rows read better as separate smaller tables

The fix is formatting, not substance.
