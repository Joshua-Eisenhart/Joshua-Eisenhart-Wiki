# Research Index

## Overview
This page is the searchable index for the compression and systems-math term family. It is not the paper/source coverage ledger; for downloaded/citation-only research-source state use [[research-source-coverage-index]] and [[notebooklm-reference-pack-intake]].

## Core terms
- PCA
- QPCA
- SVD
- Schmidt decomposition
- covariance matrix / covariance operator
- density matrix / density operator
- low-rank approximation
- entanglement spectrum
- principal subspace

## Promotion rule
A term belongs in this index if it can be stated as a principal-subspace, spectral-truncation, or low-rank operator problem.
