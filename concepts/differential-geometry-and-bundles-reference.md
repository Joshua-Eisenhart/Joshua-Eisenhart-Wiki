# Differential Geometry And Bundles Reference

## Overview
Reference doc covering differential geometry and fiber bundles concepts relevant to the system. External tradition -- actual formal concepts, not the system's interpretation.

## Key Concepts

### Fiber Bundles
A fiber bundle is a space E (total space) with a projection pi: E -> B to a base space B, such that locally E looks like B x F (the fiber). The system's Hopf structure (S^3 -> S^2 with fiber S^1) is a principal U(1) bundle. See [[hopf-fibration-mathematics]] for the specific system treatment.

### Connections
A connection on a bundle defines a notion of "horizontal" vs "vertical" transport. The Hopf connection A = dphi + cos(2eta)dchi separates fiber loops (density-stationary) from base loops (density-traversing). This is why Berry phase and holonomy appear in the system.

### Curvature
The curvature of a connection measures how parallel transport around a closed loop fails to return to the starting point. Berry curvature is the physical manifestation. See [[berry-phase-and-holonomy]].

### Parallel Transport
Moving a vector/section along a path while keeping it "constant" with respect to the connection. Fiber loops in the Hopf structure are parallel transport along the fiber -- they return to the same density matrix. Base loops traverse different densities.

### Riemannian Metrics on State Space
The system uses several metrics: trace distance (operational), Bures distance (information-geometric), Fubini-Study (projective), quantum Fisher information (sensitivity). These are NOT interchangeable. See [[cross-domain-equivalence-map]] for how they relate.

## Connection 1-forms and curvature 2-forms (formal treatment)

### Principal bundle setup
A principal G-bundle P(M, G, pi) has total space P, base M, structure group G acting freely from the right. At each p in P, the tangent space splits into vertical V_p = ker(d_pi) (tangent to fiber) and horizontal H_p (chosen complement depending on connection).

### Connection 1-form
A connection is a Lie algebra-valued 1-form omega in Omega^1(P, g) satisfying:
- Vertical reproduction: omega(xi^#) = xi for all xi in g
- Equivariance: R_g^* omega = Ad(g^{-1}) o omega

The horizontal distribution is H_p = ker(omega_p). In local trivialization: omega = g^{-1}dg + g^{-1}Ag where A is the gauge potential.

References: Kobayashi & Nomizu, "Foundations of Differential Geometry" Vol. 1 (1963); Nakahara, "Geometry, Topology and Physics" Ch. 10 (2003).

### Curvature 2-form (Cartan structure equation)
The curvature is Omega = d omega + [omega, omega] = d omega + omega wedge omega.

Properties:
1. Horizontal: Omega(X,Y) = 0 if X or Y is vertical
2. Equivariant: R_g^* Omega = Ad(g^{-1}) o Omega
3. Bianchi identity: D Omega = d Omega + [omega, Omega] = 0

Local form: Omega = g^{-1}F g where F = dA + A wedge A.

### Holonomy and the Ambrose-Singer theorem
The holonomy map Hol(gamma) for closed loops gamma at x is right multiplication by hol(gamma) in G. The Ambrose-Singer theorem (1953): Lie(Hol_x) = span{Omega_p(X,Y) : p in P, X,Y in H_p}. This is the fundamental link between curvature and holonomy -- the holonomy algebra is generated by curvature values.

## Chern-Weil theory and characteristic classes

### Chern-Weil construction
Given a principal GL(n,C)-bundle with connection omega and curvature Omega:
1. Choose an invariant polynomial f on gl(n,C): f(Ad(g) X) = f(X)
2. Apply f to curvature: f(Omega) in Omega^{2k}(M)

Chern-Weil theorem:
- f(Omega) is closed (follows from Bianchi)
- [f(Omega)] in H^{2k}(M; R) is independent of connection choice
- The map f -> [f(Omega)] gives a homomorphism (Sym(g*))^G -> H*(M; R)

References: Chern, "Characteristic classes of Hermitian manifolds," Annals of Math. 47 (1946); Weil, "Geometrie differentielle des espaces fibres" (1950); Milnor & Stasheff, "Characteristic Classes" (1974); Bott & Tu, "Differential Forms in Algebraic Topology" (1982).

### Chern classes
For a complex vector bundle E of rank n with curvature Omega:

Total Chern class: c(E) = det(I + (i/2pi)Omega) = 1 + c_1(E) + c_2(E) + ... + c_n(E)

Explicit formulas:
- c_1(E) = (i/2pi) Tr(Omega) -- first Chern class
- c_2(E) = (1/(2pi)^2) 1/2 [Tr(Omega)^2 - Tr(Omega^2)] -- second Chern class

For line bundles: c_1(L) completely characterizes L up to isomorphism.

### Chern-Simons forms
When connections vary: f(Omega_1) - f(Omega_0) = d[CS(f; omega_0, omega_1)]. The Chern-Simons form satisfies d CS(omega) = f(Omega). For 3-manifolds: CS(omega) = (1/8pi^2) Tr(A wedge dA + 2/3 A wedge A wedge A). Directly relevant to topological quantum field theory.

## Berry phase as holonomy (external tradition)

### Berry connection and curvature
For a quantum system with Hamiltonian H(R) depending on parameters R in M, and non-degenerate eigenstate |n(R)>:

Berry connection: A_mu = i<n(R)|partial_mu|n(R)> -- a U(1) connection on the line bundle L_n -> M.

Berry curvature: F = dA, with F_munu = -2 Im sum_{m!=n} <n|partial_mu H|m><m|partial_nu H|m> / (E_n - E_m)^2.

Berry phase for adiabatic evolution around closed loop gamma: gamma_B = oint_gamma A = integral_S F (Stokes theorem).

Key: gamma_B is geometric -- depends only on path topology in parameter space, not dynamics.

References: Berry, Proc. R. Soc. Lond. A 392 (1984); Simon, Phys. Rev. Lett. 51 (1983); Wilczek & Zee, Phys. Rev. Lett. 52 (1984).

### Non-abelian Berry connection (Wilczek-Zee)
For degenerate eigenspaces of dimension d, the Berry connection becomes u(d)-valued: (A_mu)_ab = i<n_a|partial_mu|n_b>. Curvature is non-abelian: F_munu = partial_mu A_nu - partial_nu A_mu + [A_mu, A_nu]. Holonomy is a unitary matrix U in U(d) -- non-abelian holonomy on a principal U(d)-bundle.

### Berry curvature and Chern classes
For a family of Hamiltonians parametrized by a closed 2-manifold M^2:
c_1 = (1/2pi) integral_{M^2} F

The first Chern number equals the quantized Berry flux. Examples: Haldane model (c_1 = +/-1 per band), spin-1/2 in magnetic field on S^2 (c_1 = 2S, monopole charge), TKNN invariant (Hall conductivity sigma_xy = (e^2/h) sum_n c_1^{(n)}).

### Quantum geometric tensor
The QGT combines Berry curvature (antisymmetric) and Fubini-Study metric (symmetric):
Q_munu = <partial_mu psi|(1 - |psi><psi|)|partial_nu psi> = g_munu + (i/2)F_munu

where g_munu = Re(Q_munu) is the Fubini-Study metric (= quantum Fisher information) and F_munu = -2 Im(Q_munu) is the Berry curvature. This gives (M, g, omega) an almost-Kahler structure (Kahler when complex structure from Hilbert space is compatible).

References: Provost & Vallee, Commun. Math. Phys. 76 (1980); Kolodrubetz et al., arXiv:1610.08991 (2016).

### Chern numbers as topological invariants
- First Chern number (1/2pi) integral F: quantized Hall conductance, band topology
- Second Chern number (1/8pi^2) integral Tr(F wedge F): second-order nonlinear response, 4D quantum Hall (Zhang-Hu, Science 294, 2001)
- Chern-Simons 3-form: topological invariant of 3D systems, TQFT

The tenfold way (Altland-Zirnbauer classification) maps to K-theory groups computed via characteristic classes. See Kitaev, AIP Conf. Proc. 1134 (2009) for the periodic table.

## Mismatch Notes
Standard differential geometry assumes smooth manifolds everywhere. The system operates on finite-dimensional quantum state spaces (D(C^2)), which are not smooth in the classical sense. The "geometry" is operator-algebraic, not Riemannian in the classical sense. The Chern-Weil framework still applies via the operator-algebraic curvature, but the physical interpretation differs from the classical manifold case.
