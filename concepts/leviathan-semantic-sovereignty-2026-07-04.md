## Overview

Leviathan's name comes from Hobbes. Owner (2026-07-02): "at root leviathan controls semantics. its very name comes from hobbes leviathan. the central power of the sovereign is defining words. all shared connections of people and even avatars is governed by a shared dictionary. a mimetic meme manifold — MMMs. the very dictionary ontology and semantics creates the very governance of reality. all social matrixes nested down are shared dictionaries."

## Hobbes: the sovereign's core power is defining words

In Hobbes's Leviathan, the sovereign's central power is fixing the meaning of words — settling disputes over language so that shared action is possible at all. Applied here: the MMM is that shared dictionary. It is the object that governs all connections between people and between agents ("avatars"), because shared connection requires shared semantics. Nested social matrices, all the way down, are shared dictionaries.

## Implication: vocabulary changes are gated constitutional acts

Because the dictionary governs the connections, changing the dictionary changes governance. Dictionary changes are therefore constitutional acts: they must be versioned, gated, and sovereign-scoped. Each node owns its own lexicon; boundary vocabulary is jointly held between nodes, not owned by one side.

## Dawkins memetics, done rigorously

This is where the MMM completes the Leviathan framing rather than duplicating it. Dawkins applied the replicator principle three ways: genes, reality (the owner's cosmology extension — see the teleological selection material), and memes. The meme application is Leviathan's territory, and MMM literally names it: Mimetic Meme Manifold.

Owner (2026-07-03): memetics is Dawkins's weakest application — fuzzy copy-fidelity, an ill-defined unit, a meme that mutates on every retelling. That is the standing scientific objection to memetics.

Leviathan repairs it. A typed, content-addressed, versioned MMM entry is a discrete meme with a checkable copy — the gene-like discreteness that wild memes lack. The content-address is the copy-fidelity mechanism: two nodes carry the same meme if and only if they share the same hash, not if it merely "feels similar." The gate is the selection operator on the meme pool.

Dawkins's three requirements become concrete:

- heredity = content-addressed copy
- variation = mutation on retry and rework (see [[purgatory-working-space-2026-07-04]])
- selection = the gate

Hobbes loop: semantic sovereignty is governing which memes replicate. The sovereign is the gate on the meme manifold. Whoever gates the MMM controls the culture's heredity.
