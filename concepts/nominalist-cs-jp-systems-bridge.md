# Nominalist-CS and JP Systems Terms Bridge

*Religious/archetypal proper nouns below are metaphorical handles for structural claims. Neither load-bearing in the formalism nor removable — they do the communicative work. This is a translator, not an endorsement or a reduction.*

## Thesis

Jordan Peterson's popular vocabulary (order/chaos, Logos, archetype, hero, dragon, Christ-as-pattern) describes the same constraint-survival dynamic this project formalizes as probe-relative constraint-admissibility. The bridge is structural, not doctrinal. JP leans Platonist; this framework is nominalist. The tension is real and is held, not resolved.

## Why JP Is Structurally Nominalist Despite Platonist Rhetoric

1. Archetype as projectible predicate. JP's operational archetype is a cross-population, cross-epoch probe-survivor — a Goodman projectible pattern, not a transcendent form. Real as material statistical invariant; not real as separate substance.
2. Logos as generator, not substance. "Logos carves order from chaos" = there is a rule-set under which chaos quotients into stable distinctions. That rule-set is F01+N01 here. JP slides between Logos-as-process and Logos-as-Being; the nominalist reading keeps the first and refuses the second.
3. Hero's journey is exclusion-survival. No new substance is generated; an already-candidate object is promoted by surviving named kill conditions. That is admissibility, not construction.

The tension is not papered over. JP's preferred reading is Platonic; this project's formalism rejects that reading. The bridge is honest only because it names the disagreement.

## "JP As Architect Of God" — Structural Reading

Metaphor. The structural claim underneath:

- JP is constructing a durable narrative harness that re-finds the same admissibility pattern across substrates.
- The "god" at the terminus is a metaphor for the backward-admissibility invariant — the pattern re-findable from scratch by a future population after reset.
- This matches the AI-legacy durability test in [[self-similar-frameworks-and-teleological-doctrine]]: the pattern must be re-findable with none of today's references.

No substance claim is being made. The structural claim is that JP's rhetorical target and this project's backward-admissibility arrow point at the same feature of constraint-survival geometry.

## Operational Rules

1. Do not import JP's Platonism into the formalism. Flag "the archetype IS real" / "Logos is the ground of Being" as the target bias.
2. Do not strip JP's vocabulary from outward-facing docs — it does communicative work. Mark as metaphor; never load-bearing.
3. Translate bidirectionally: formal claim → CS → JP handle for outward writing; JP term → CS → formal for inward reading.
4. Never cite Christ / God / Logos as primitives in a sim. Fails the load-bearing-tool gate.
5. Hold the tension. Docs that resolve JP's Platonism into this nominalism are lying. Docs that hold the disagreement are doing the work.
6. "Dominance hierarchy" = probe lattice, not sociology. Do not let JP's social deployments leak.
7. Hero's journey = Coupling Program Order. Dragon = named kill condition at each stage.
8. Shadow discipline: summaries omitting exclusion history perform the shadow-denial anti-pattern.
