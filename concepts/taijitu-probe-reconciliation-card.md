## Overview
This digest keeps what the taijitu symbolic layer can say safely, what it cannot say safely, and where it remains useful as a witness/crosswalk surface instead of a math anchor.

## Current strongest alignments
- `Axis 0`: black-versus-white is still a usable symbolic split, but it does not close the `Phi_0(rho_AB)` bridge or the cut-state family.
- `Axis 1`: dot-and-teardrop symbolism remains a usable witness for the branch split `{ Se, Ni }` versus `{ Ne, Si }`.
- `Axis 2`: dots-versus-teardrops can still witness the direct-frame versus conjugated-frame split, but the lower-stack math anchor remains primary.
- `Axis 3`: inner-versus-outer loop class is stronger than the older left-versus-right chirality language.
- `Axis 6`: judging-first versus perceiving-first order is directly readable from the ordered-token partitions and is stronger than older boundary-side phrasing.
- The 64-slot scaffold still works as an addressing surface if it stays fenced as representation rather than doctrine.

## Active mismatch fences
- `Axis 0`: symbolic black/white language must not replace the lower-stack bridge/cut family.
- `Axis 1`: older interpenetration/coherence language is weaker than the current derived branch split.
- `Axis 2`: dots-versus-drops or concentrated-versus-spread language is weaker than direct-versus-conjugated frame.
- `Axis 3`: mirror-flip or chirality-only wording is weaker than fiber-versus-lifted-base loop class.
- `Axis 4`: clockwise/counterclockwise stays symbolic only; the stronger runtime anchor is `UEUE` versus `EUEU`.
- `Axis 5`: S-curve and lobe-shape language stays symbolic only; the stronger runtime anchor is the operator-family split `T = { Ti, Te }` versus `F = { Fi, Fe }`.
- `64_HEXAGRAM_STATE_SPACE.md` still carries stale line meanings for `Axis 3`, `Axis 4`, and `Axis 5`; use it only as a fenced addressing scaffold.
