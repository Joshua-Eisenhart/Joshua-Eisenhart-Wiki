# FEP Glossary

Vocabulary support for the FEP atlas. Definitions here are routing aids, not local evidence.

| Term | Working meaning | Guardrail |
|---|---|---|
| variational free energy `F` | objective for current inference/update | not policy selection `G` |
| expected free energy `G` | future/counterfactual policy-evaluation quantity | not the ratchet objective |
| surprise | self-information, often `-ln P(y)` | not colloquial surprise |
| variational density | approximate belief/recognition density | not truth |
| generative model | model over hidden causes and observations | not a proven holodeck implementation |
| active inference | perception/action as inference under free-energy minimization | not Leviathan implementation |
| epistemic value | information-gain value of action | not evidence without artifact |
| pragmatic value | preference-satisfaction value of action | not Codex Ratchet teleology |
| sophisticated inference | planning with beliefs about future beliefs | finite-horizon analogy only |
| belief-state tree search | search over possible future belief distributions | not complete search of `M(C)` |
| Markov blanket | conditional-independence boundary mediating internal/external states | not Axis0 cut |
| NESS | nonequilibrium steady state | not local survival proof without dynamics |
| particular partition | internal/external/sensory/active partition | not automatically physical boundary |
| solenoidal flow | circulation-like flow in some FEP decompositions | critique-sensitive |
| natural gradient | steepest descent in information geometry | not QIT admission |
| information length | trajectory length in distribution space | not proof of metabolic or ratchet cost |
| viability kernel | states from which constraint-respecting evolution exists | must be model-defined |
| autopoiesis | self-production/self-maintenance concept | conceptual support only |
| enactivism | cognition as world-coupled activity | not formal proof |
| sense-making | organism/system-relative significance | not consciousness proof |
