# Belief-State Tree Search and Finite Horizons

Sophisticated inference is useful here because it shifts planning from future world states to future belief states.

```text
ordinary planning:
  what might happen if this action is taken?

sophisticated inference:
  what would the agent later believe would happen after acting, observing, and updating?
```

## Crosswalk

| Active-inference term | Codex Ratchet support analogy | Blocked overclaim |
|---|---|---|
| belief-state tree search | finite exploration of candidate surfaces | not complete search of `M(C)` |
| epistemic affordance | probe that changes the boundary map | not evidence without artifact |
| horizon depth | how far the system evaluates consequences | not license for ungated speculation |
| local minimum | shallow planning trap | not proof of Axis0 or holodeck |
| recursive expected free energy | beliefs about future beliefs | not root-constraint derivation |

## Finite-horizon discipline

Finite search is not a weakness to hide. It is the correct bridge to F01-style discipline:

- name the state representation;
- name the action/probe set;
- name the horizon;
- name the update rule;
- record what was not searched;
- keep every result at its earned ceiling.
