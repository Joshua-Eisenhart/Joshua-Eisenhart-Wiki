# Current-Edge Library Backlog for Codex

The current-edge set is a candidate menu, not an install order.

Best first additions:

- `Catlab.jl + AlgebraicRewriting.jl` for typed C-sets, graph/hypergraph rewriting, DPO/SPO-style probes.
- `XGI / HyperNetX / hypergraphx` for higher-order network scouts.
- `egg / egglog` for quotient/equivalence/rewrite saturation.
- `CombinatorialSpaces.jl + GUDHI + TopoNetX` for topology and DEC checks.
- `rustworkx` for fast DAG/event/multiway graph work.
- `torch_geometric` for load-bearing graph/network dynamics only when ablation changes the observable.

Every library needs support + micro-probe receipts before load-bearing use.
