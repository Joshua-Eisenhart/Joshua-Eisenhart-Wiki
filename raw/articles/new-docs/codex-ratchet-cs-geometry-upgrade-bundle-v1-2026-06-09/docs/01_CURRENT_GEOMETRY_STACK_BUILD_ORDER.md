# Current Geometry Stack Build Order

Foundation-first geometry, not novelty chasing.

1. Root constraints: finite carrier/probe/operator/path set and order-sensitive control.
2. M(C): state set, constraints, quotient/readout map, admissibility predicate, composition/bracketing, controls, receipt paths.
3. Carrier/readout base: C^2, density states, Pauli/Hermitian probes, quotient S/~_M, carrier-vs-readout erasure controls.
4. Spinor / Clifford carrier: same finite carrier, spinor lift, Clifford action, gamma structure.
5. Hopf / S^3 -> S^2 / fiber-base geometry: fiber loop vs base loop, Hopf connection, nested Hopf tori.
6. Weyl / chirality layer: L/R sheets, gamma7 split, chiral density bookkeeping, sign-erasure controls.
7. Path / holonomy / order layer: SU(2) spinor holonomy, loop order, reverse controls.
8. Bracketing / nonassociativity: associator, octonion/Cl carrier pressure, division-algebra boundary.
9. Multi-spinor / network / cut-state precursor: PyTorch/PyG only if ablation changes the observable.
10. Metric / information geometry: Fubini-Study, Bures, QFI/QGT, trace distance, geomstats.
11. Topology / graph / cell-complex layer: GUDHI, TopoNetX, XGI, rustworkx, persistence, incidence, orientation.
12. Terrain / operator / engine grammar: maps from M(C), terrain/operator tokens, UP/DOWN precedence.
13. Attractor-basin layer: only after finite state/update/boundary/invariant/escape controls.
14. Cross-model readout matrix: QIT / IGT / Holodeck / physics lenses over same support.
15. Entropy / memory / viability: later readout pressure only.
16. Physics-facing metrics: fenced as math-only controls until lower object admitted.
