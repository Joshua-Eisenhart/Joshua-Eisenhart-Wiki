# Mimetic Meme Manifold Source Digest

## Repo-backed language facts
- boot prompts are the harness for a terminal
- multiple boots can run simultaneously
- each boot role has its own vocabulary and salience profile
- A2 builds/routers the harness, A1 explores, B enforces, A0 compiles, SIM executes
- multiple saliences may coexist, but they must not contaminate each other
- boot prompts need concrete templates

## Working interpretation
A harness-language dictionary is a salience-shaping layer that keeps a model near the system’s primitives.

## Local term
"Mimetic Meme Manifold" = the structured language field that biases a model toward the system’s real constraints and boot roles.
