- density matrices unify state, measurement, and evolution
- entropy families have quantum/operator versions (vN, Rényi, Tsallis, min/max, conditional, mutual, coherent info)
- compression math maps to spectral/operator structure (SVD, PCA, Schmidt, low-rank)
- geometry maps to state-space geometry (trace distance, Bures, Fubini-Study, QFI, QGT, Berry)

## Fields touched
- quantum computing
- physics
- computer science / algorithms
- AI / ML
- evolutionary / selection-style dynamics

## Working hypothesis
Many systems can be reframed as state + probe + constraint manifold + admissible geometry + information functional + evolution rule.
