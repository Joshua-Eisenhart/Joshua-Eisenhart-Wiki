# Phrasebook — Classical → Nominalist Clause Rewrites

Extends the 10-row rewrite table in `03_language_discipline.md`. Verb swaps alone do not catch clause-level smuggling.

## Existence and identity

| Classical | Nominalist |
|---|---|
| X exists | X is admissible under constraint set C |
| X is real | X survived probes in M under C |
| X is the same as Y | X is indistinguishable from Y under M |
| X equals Y | X ~_M Y |
| there is an X | candidate X is admissible under C |
| the X | the admissible X under C (or cite M) |
| X and Y are identical | no probe in M distinguishes X from Y |
| X itself | X as a survivor class under M |

## Properties and attributes

| Classical | Nominalist |
|---|---|
| X has property P | probe p ∈ M returns P on X |
| X is P | under probe p, X returns P |
| property P of X | output of probe p on X |
| X is inherently P | under every probe in M, X returns P |
| X really is P | X returns P under every probe tested; alternatives not yet excluded |

## Causation and generation

| Classical | Nominalist |
|---|---|
| X causes Y | Y is coupled with X under dynamics D |
| X produces Y | Y survives under the constraint that includes X |
| X generates Y | Y is admissible in the presence of X |
| X makes Y happen | Y co-varies with X under M |
| X leads to Y | the branch containing Y is consistent with X |
| X forces Y | configurations without Y are excluded under C when X is active |
| because X, Y | Y is consistent with X; other admissible explanations remain |
| X is why Y | Y is coupled with X; exclusion of other couplings is incomplete |

## Proof and truth

| Classical | Nominalist |
|---|---|
| this proves X | this configuration survived the constraints tested |
| X is true | X is admissible under current C and M |
| X is false | X is excluded under C (or: UNSAT under C, if formal) |
| X is correct | X is admissible; alternatives not yet excluded |
| X is wrong | X was excluded by probe p |
| we confirmed X | X survived rerun under C on `date`; result at `path` |
| this proves X is fundamental | X was load-bearing in every probe tested so far |
| X must be the case | X is the only surviving candidate under current C |

## Knowledge and belief

| Classical | Nominalist |
|---|---|
| I know X | X is admissible under C; I can cite the probe |
| I think X | candidate X is in the surviving set; other candidates remain |
| X is likely | X is consistent with current evidence; exclusion of alternatives incomplete |
| X probably holds | X survives; alternatives have not been excluded |
| it's obvious that X | X is admissible under minimal probes in M |
| everyone knows X | X is admissible under standard probe families; cite the family |
| we don't know | no probe in M currently distinguishes the surviving candidates |

## Description and characterization

| Classical | Nominalist |
|---|---|
| X is a kind of Y | X sits in the survivor class of Y under M |
| X belongs to category C | X is admitted by the probes defining C |
| the nature of X | the behavior of X under probe family M |
| X's essence | the invariant of X across probes in M |
| X is fundamentally Y | X returns Y under every probe tested; alternatives not excluded |
| the structure of X | the quotient S/~_M restricted to X |

## Process and dynamics

| Classical | Nominalist |
|---|---|
| X evolves into Y | under dynamics D, the branch containing X reaches Y |
| X transforms into Y | X and Y are coupled under D; D maps X to admissible Y |
| X becomes Y | X ~_M Y after dynamics D (or: Y is the admissible successor) |
| X emerges from Y | X is admissible in configurations where Y is active |
| X depends on Y | X co-varies with Y under M |
| X requires Y | no X survives C without Y; exclusion under ¬Y is UNSAT |

## System-level

| Classical | Nominalist |
|---|---|
| the correct density matrix | the admissible density matrix under C |
| the true geometry | the geometry that survived all simultaneous constraint layers |
| the right answer | the surviving candidate under C; others not yet excluded |
| the fundamental equation | the equation load-bearing in every probe tested so far |
| the underlying reality | the support geometry at the highest active constraint |
| the actual state | the admissible state under C given M |

## Sim-report phrases

| Classical | Nominalist |
|---|---|
| all tests pass | 28/28 tests passed under C on `date`; result at `path` |
| verified | passed local rerun on `date`; criteria: [...]; result at `path` |
| the sim works | the sim exists / runs / passes local rerun / is canonical by process (pick one, cite) |
| confirmed the hypothesis | configuration survived the constraints; alternative candidates remain |
| proves the model | the model survived probes p_1 ... p_n; probes p_{n+1} ... remain untested |
| all pass | specify criteria + cite result file path (see status ladder) |

## Use

When drafting, scan for a classical phrase from the left column. Substitute the right column. If a right-column phrase feels "too weak", that is the harness working — the classical phrase was claiming more than the artifact surface earned.
