# Sophisticated Inference and Leviathan

Sophisticated inference is useful for Leviathan because it talks about planning over future belief states, not just future external states.

```text
ordinary planning:
  what might happen if I act?

sophisticated inference:
  what will I believe after I act, observe, update, and plan again?
```

That is a useful external comparison lane for Leviathan's world-model, reconciler, graph/context, and policy/control directions.

## Crosswalk

| Sophisticated-inference object | Leviathan neighbor | Guardrail |
|---|---|---|
| future belief state | graph/context/world-model state after action and observation | not proved as implemented |
| recursive planning | planner/reconciler loop | needs source and command receipts |
| epistemic affordance | action that reduces uncertainty | not evidence unless a flow emits artifacts |
| finite horizon | bounded planning depth | not universal reasoning |
| local minima | shallow policy trap | not Axis0/Holodeck proof |

## Where To Use It

Use this page when writing about:

- world-model/reconciler directions;
- policy or gating decisions under uncertainty;
- context reconciliation;
- event traces as observation/action histories;
- future validation fixtures.

## Where Not To Use It

Do not use sophisticated inference to claim:

- Leviathan already implements active inference;
- the world-model package gates production decisions;
- Codex Ratchet is derived from Leviathan;
- QIT, Axis0, Holodeck, gravity, or consciousness claims are supported.
