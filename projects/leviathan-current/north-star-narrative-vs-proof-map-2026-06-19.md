# Leviathan North Star Narrative vs Proof Map

`docs/NORTH_STAR.md` is valuable for product category and narrative. It is not a proof receipt.

## Safe use

- product positioning;
- category thesis;
- AgentPing / AgentGuard / FlowMind relationship;
- Joshua constraint attribution;
- constraint manifold story;
- world-model narrative.

## Unsafe use

- proving tests pass;
- proving enterprise readiness;
- proving world-model production wiring;
- proving every listed feature is productized;
- settling release readiness.

## Safe posture

North Star can explain what Leviathan wants to be. Current proof pages decide what Leviathan can presently claim.
