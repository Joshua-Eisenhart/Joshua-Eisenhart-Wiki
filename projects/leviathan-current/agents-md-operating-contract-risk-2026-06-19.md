# Leviathan AGENTS.md Operating Contract Risk

`AGENTS.md` is a force multiplier and a risk surface. It is an operating contract for coding agents, not product truth.

## Strengths

- evidence-over-assumption rule;
- autonomous completion posture;
- small reversible diffs;
- no new dependencies without request;
- lint/typecheck/test/static analysis expectations;
- lore commit protocol;
- delegation rules and team-mode boundaries;
- package API rules against sloppy re-exporting;
- model-routing and skill-trigger conventions.

## Risks

- strong autonomy language can push agents to over-act when scope is unclear;
- keyword auto-triggering can activate unexpected workflows;
- long dense rules invite partial reading;
- agent work discipline can be mistaken for runtime/product proof;
- operating culture may sound greener than command evidence supports.

## Safe claim

AGENTS.md gives Leviathan a serious agent-work operating contract, but it is not evidence that the product runtime is green.
