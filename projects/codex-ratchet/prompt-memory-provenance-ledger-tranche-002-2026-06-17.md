# Prompt Memory Provenance Ledger Tranche 002 — Claude CR + Claude Desktop CR

## Purpose

This tranche processes a bounded set of Claude project-memory and Claude Desktop project-memory rows into safe wiki routing. It is not a dump of memory files and not a promotion of any remembered claim into canon.
