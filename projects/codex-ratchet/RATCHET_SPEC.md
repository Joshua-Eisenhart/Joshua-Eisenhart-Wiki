## 3. MSS IS DEFEASIBLE — every rung is PROVISIONAL (owner, 2026-07-10)
**MSS (Minimal Survivable Structure):** admit ONLY the weakest structure that (a) survives the active constraints and (b) is still
killable/composable under a *supplied* update family. Nothing stronger is admitted unless a lower closure FORCES it.

**The provisionality law (state it at every rung):**
- One can NEVER prove there is no weaker MSS candidate. A weaker structure that carries the same distinction may always be proposed.
- **Every earned rung is PROVISIONAL on no better (weaker) MSS candidate being found that still works.** An earned rung is
  therefore DEFEASIBLE: a later weaker survivor DEMOTES it. This is not a flaw — it is the ratchet's honesty condition.
- **The weakness-ordering itself is INSTALLED** (fleet 7/7, wf w1a8hu1g0), not forced by probe data. Name it as installed.
- **MSS smuggles dynamics:** "survives/evolves" is sound only if read as "killable/composable under a SUPPLIED update family `U`,"
  never "generates its own dynamics." The ratchet's order + updates + memory are **supplied structure on the forced static floor.**
- **Forced vs installed, per rung** (fleet-grounded): quotient `Q=X/~_P` = FORCED; `ω`/non-commutativity/`ρ`/complex/Hilbert =
  INSTALLED (a second non-isomorphic carrier gives the same quotient behavior ⇒ installed). The discriminator: `S` is FORCED only
  if EVERY admissible extension compatible with the same probe table + supplied constraints is isomorphic at the observable quotient level.

## 5. THE RATCHET LOOP (every step has exactly this form)
```

3 Weakest structure least-assumptive structure carrying the distinction
4 Constraint        which probe/order/bracket/cut tests it?
```

State object `R_t = (X_t, P_t, Q_t, G_t, E_t, H_t)`. Update: carve `X_{t+1}` by admissibility, then **recompute `Q,G,E` on the
ALTERED survivor set** (geometry recomputes after each carve; the next constraint acts on the altered geometry, not the free one).
Minimalist agent goes FIRST ("can the weaker structure do this?"); only on "no" does the Lift-Advocate argue; then the
Control-Auditor tries to kill the lift.
