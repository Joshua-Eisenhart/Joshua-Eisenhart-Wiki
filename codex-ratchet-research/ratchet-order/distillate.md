### Naming discipline

- `monoid_action_row` is a label for a finite action model, not proof that the
  Ratchet system is a monoid.
- `inverse_semigroup_row` is a label for partial-symmetry tests, not proof that
  all operations have inverse-semigroup structure.
- `quotient_tower_row` is a label for typed quotient maps, not an abstraction
  ladder with automatic authority.
- `bracketing_row` is a label for a tested source/target-typed associator
  witness, not a claim that parentheses create physics.
- `normal_form_row` is a label for a canonicalization attempt, not proof that a
  canonical form exists.
- `sheaf_obstruction_row` is a label for a fully typed local/global test, not a
  metaphor for "many perspectives."

### One-line bounded distillate

Ratchet-order language should be exclusion-first: names are labels, quotients
must say what they forget, normal forms and coherence kill alias-inflated path
claims, and order identity remains a finite test result rather than an assumed
object.
