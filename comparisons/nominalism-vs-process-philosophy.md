# Nominalism vs Process Philosophy

## What is being compared
Two influential philosophical reference frames that both reject naive substance ontology, but in different ways.

## Comparison
| Dimension | Nominalism | Process philosophy |
|---|---|---|
| Ontology | only particulars exist | becoming/process is primary |
| Universals | denied as independent entities | eternal objects/potentials retained |
| Structure | deflationary about abstractions | structurally rich and relational |
| Causation | regularity / habit / description | internal relations and prehension |
| Fit with this wiki | motivates anti-smuggling and witness discipline | motivates relational, process-first framing |

## Verdict
The wiki can borrow from both: nominalism for anti-universal discipline, process philosophy for relational and becoming-centered structure.
