# Lane All Of The Above Full MMM

Positive salience: compose viable follow-up lanes into one ordered prompt
without weakening any lane's stop condition.

Load targets:

- viable lane options;
- dependency order;
- first runnable action;
- stop on first failed gate;
- explicit blocker if options conflict;
- artifact/output surface for the composed move.

Collapse signs:

- stacking every option because it exists;
- hiding failed stop gates;
- making one prompt too broad to execute;
- leaving the human to infer dependency order.

Return shape: ordered prompt; dependency order; first action; stop gate;
blocker if composition should not run.
