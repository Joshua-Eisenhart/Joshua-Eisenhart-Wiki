# Support-First Preflight

(added 2026-06-03)

Purpose: before any operator / bridge / axis / manifold claim, the carrier, support, and coexistence relation must be named and cited. If the carrier/support/coexistence surface is not cited, the later claim is blocked rather than softened into prose.

This file is a preflight, not a proof surface. Passing it does not admit the later structure by itself; it only prevents support-smuggling before the relevant coupling and bounded-work gates are checked.

---

## Admission Rule

A sentence or work unit that claims an operator, bridge, axis, or manifold structure is admissible only when it first names what the claim runs on.

Required shape:

```
Given carrier <K>, support relation <S>, and coexistence evidence <R or open>,
claim <X> is [exists | runs | passes local rerun | canonical by process].
```

If `<K>` or `<S>` is absent, the sentence completes as:

```
Support not established; operator / bridge / axis / manifold claim blocked.
```

---

## Checklist

1. Carrier named.
2. Support relation cited.
3. Coexistence evidence cited or marked as open.
4. Later operators blocked until support is established.

Any missing item blocks the later claim. Do not replace the block with "likely," "natural," "emergent," "obvious," or narrative continuity.

---

## Failure Modes

- **Carrier omission**: an operator or bridge is described without naming the state space, shell, carrier tensor, substrate, or other support it runs on. Intercept: name carrier first or block.
- **Support by implication**: the sentence assumes a manifold, topology, or axis label already supplies support. Intercept: cite the support relation separately from the label.
- **Coexistence laundering**: pairwise or shell-local evidence is used as if simultaneous coexistence had already survived. Intercept: cite coexistence evidence or mark coexistence open.
- **Later-operator leakage**: bridge / axis / manifold language appears before support is established. Intercept: stop at the support preflight; downstream claim is a separate bounded work unit.
