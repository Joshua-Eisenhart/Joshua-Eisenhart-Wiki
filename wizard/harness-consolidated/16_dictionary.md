# System Dictionary

Compact routing note for the system's key nouns. The detailed glossary was archived to `wiki/_archive/harness_2026-04-17_steward_trim/16_dictionary.md` so the live harness stays within primer bounds.

Use these anchors:
- identity -> probe-relative equivalence class, never primitive self-sameness
- probe family `M` -> finite resolution surface that licenses distinctions
- admissibility -> survivor status under active constraints
- candidate -> provisional survivor, not closure
- exclusion -> strongest narrowing form; UNSAT is the clean certificate surface
- quotient -> ontology at the active probe resolution
- form -> material pattern that survives probes and constraints
- shell / manifold support -> local support surface that must be earned in the lane, not silently imported

Write discipline:
- when a noun would otherwise import a classical default, restate it with probe and constraint context
- if a term cannot be grounded through probe family + admissibility + quotient structure, treat it as unsafe shorthand and expand it
- use `18_red_team_probes.md` to test whether the compact grounding held
