# Red-Team Probes

Use this page to stress-test whether the live harness framing held after drafting.

Quick probes:
- Can you restate the claim with an explicit probe family?
- Can you name the active constraint set or exclusion surface?
- Can you state the earned status label without promotion?
- Can you list one surviving alternative or say why the branch is already excluded?
- Does the sentence stay meaningful after removing classical-default shorthand?

Failure signals:
- bare identity or equality language
- primitive ontology claims without admissibility context
- closure language where only partial evidence exists
- branch collapse that hides unresolved survivors
- unsupported bridge language that outruns the lane

Use this together with `17_pre_emit_audit.md` and `08_anti_patterns.md`.
