# Phrasebook — Clause Rewrites

Extended pre-trim version lives at `wiki/_archive/harness_2026-04-17_steward_trim/20_phrasebook.md`. This live page keeps the rewrite surface compact.

Use these swaps when a sentence quietly imports primitive ontology or premature closure.

## Core swaps

- bare existence -> admissible under `C`
- bare identity / equality -> indistinguishable under `M`
- property talk -> probe `p` returns ...
- essence / nature -> invariant across probes in `M`
- single-winner wording -> surviving candidate under current `C`
- certainty words -> cite the earned status label and result path
- hidden dependency story -> use `coupled with`, `co-varies under`, `consistent with`, or `excluded without Y`
- proof / truth talk -> `survived the constraints tested`, `excluded under C`, or `UNSAT under C`
- guess-plus-closure -> keep the branch open instead of collapsing it

## Geometry / time / probability / support swaps (added 2026-06-03)

- `the geometry determines...` -> `the constraint set admits this geometry; it survived under probe M`
- `the geometry is...` -> `under constraint set C and support surface S, geometry G is admitted at status label L`
- `over time, X evolves...` -> `under forward evolution, X is reachable; backward admissibility still open`
- `X changes over time...` -> `under evolution relation E constrained by C, observable O changes relative to negative/control condition N`
- `the probability is P` -> `under the constraint-compatible measure, the admissible weight is P; no primitive probability is claimed`
- `the probability of X...` -> `within measure mu_C over candidate set S, X has admissible weight P; alternate measure choices remain named if open`
- `the bridge connects...` -> `the bridge is admissible given carrier C and support S; without S the claim is blocked`
- `the operator acts on...` -> `given carrier K and support relation S, operator O is defined on domain D at status label L`
- `Axis A has manifold structure...` -> `Axis A remains a hypothesis-space label until carrier K, support S, and coexistence evidence R are cited`
- `the manifold supports...` -> `support relation S on carrier K admits claim X under C; no primitive manifold support is claimed`

## Sim-report replacements

- `verified` / `all pass` -> state the actual ladder label, date, criteria, and file path
- `the sim works` -> choose one: exists / runs / passes local rerun / canonical by process
- `proved the model` -> list the probes survived and the probes not yet run
- `the correct X` / `the true X` -> `the admissible X under C` or `the current survivor set`

## Safe clause templates

- `Under probe family M, X returns P.`
- `Under constraint set C, X survived and Y remained open.`
- `No probe in M distinguished X from Y.`
- `Formal check: UNSAT under C.`
- `Result surface: path/to/result.json; status: passes local rerun.`
- `Under constraint-compatible measure mu_C, candidate set S has admissible weight P; no primitive probability is claimed.` (added 2026-06-03)
- `Given carrier K and support relation S, claim X is admitted at status label L; without S, X is blocked.` (added 2026-06-03)

## Use

If a rewrite feels weaker after the swap, keep the weaker form unless new evidence lands. That friction is part of the harness.
